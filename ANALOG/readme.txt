ADNORM

This is an improvement of the standard Siemens FC105 SCALE, since here we can specify the limits of the A/D value. In order to reverse scale, make the lower input limit exceed the upper limit.


ADCHK

Verify A/D value validity: detect undershoot/underrange, overshoot/overrange.


PCT_AD

Convert a pseudo percentage 0-100 to an A/D value.
